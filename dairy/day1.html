    Level 1 - Git Real
  • ‚git init‘ to add a new Repository to my Account
  • ‚git add …‘ Add my file to the repo
  • ’git log’ show commit history
    Level 2 - Git Real
  • ’git diff’ um zu sehen, welche Änderungen in einer Datei passiert sind
  • ’git reset HEAD ’, damit eine Datei nicht im stage/master committed wird
  • ’git checkout —filename’ um Modifikationen einer Datei zu resetten
  • ’git reset —soft HEAD^’ letzten Commit widerrufen, changes gehen ins staging
  • ’git commit -amend -m „New message“’ den letzten commit ändern
  • ’git push’ meine Änderungen online in remote repository
